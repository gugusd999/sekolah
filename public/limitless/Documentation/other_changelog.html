Roadmap

Road map is a kind of to-do list. It contains requests and plans for future versions. I don't specify any dates of release to avoid empty promises, but i work on this template every day, doing my best to make it perfect. Feel free to send me your thoughts, suggestions and requests to e.kopyov@gmail.com. Thanks!

In upcoming updates:

  • RTL version - right-to-left version of all layouts. Highest priority and planning to release in version 1.1.
  • Datatables - new extensions, will be included in version 1.2.1.
  • PSD guidebook - photoshop files with all styles, colors and components used in the template. Should be available in one of the next 2-3 updates.
  • Browser detection tool - a tool for OS and browser detection.
  • Mail app - custom pages set for email system: read, write and mail list.
  • Ecommerce kit - custom pages kit: checkout process, dashboard, shopping cart etc.
  • jQuery UI components - full set of jQuery UI library components.
  • New dashboards - thematic dashboards: ecommerce, corporate, analytics, minimal, billing panel and intranet.
  • Layout 5 - a new layout: tabbed interface with multi level horizontal navigation. Work already in progress, but no timeframes as i have some plans with higher priority.
  • Layout 6 - a new layout: minimalistic layout with transparent sidebars and light components. Got some really cool ideas, but need more time for polishing details.
  • Snippets - permanently updating library with pre-made widgets. Will be ready in upcoming updates.
